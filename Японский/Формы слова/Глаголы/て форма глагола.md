#### Структура
###### 五段:
**る** / **う** / **つ**  -> **って**
**く** / **ぐ** -> **いて** / **いで**
**ぬ** / **ぶ** / **む** -> **んで**
**す** -> **して**
###### 一段:
**る** -> **て**
#### Исключения
###### 五段:
1. **行く** -> **行って**
2. **する** -> **して**
3. **来る** -> **来て**（きて）
4. **問う** -> **問うて**
5. **請う** -> **請って**
