#Японский 
#### Структура
Группа элементов + **の中で** + что-то конкретное + **一番** + прилагательное для описания
#### Пояснение
Выражение, которое используется для демонстрации того, что что-то является «*самым (C)*»
#### Тонкости
#### Примеры
1. 食べ物**の中で**寿司**が一番**好き。ー　Из всей еды, моё любимое - суши.
