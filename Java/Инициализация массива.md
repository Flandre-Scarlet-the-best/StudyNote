Массивы можно инициализировать с помощью списка. При этом можно обойтись без операции **new**.
```Java
int [] b = new int[]{3, 5, 2};  
double [] c = {0.3, 2.2, 3.6, 9.2};
```
Количество элементов массива определяется  автоматически по числу значений в списке.  При инициализации списком **нельзя указывать его размер**.

Обращению к элементам идёт по индексу. Начиная с 0. Поле **length** хранит размер массива. Это поле можно прочитать, но нельзя изменить. Обычно для перебора элементов массива используется это поле
```Java
for (int i = 0; i < c.length; i++) {  
    System.out.print("|" + c[i] + "| ");  
}
```
Также для вывода массива можно использовать цикл в стиле **foreach**. В этом цикле доступ к элементам массива только в режиме чтения:
```Java
for (double element : c) {  
    System.out.print("|" + element + "| ");  
}
```
